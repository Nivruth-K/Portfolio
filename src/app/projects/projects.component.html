Interview Scheduler App:

Project Description

The Student Interview Scheduler app is a dedicated platform designed to assist students in organizing and managing their scheduled interviews within an academic or institutional setting. The app provides students with a convenient interface to keep track of interview dates, times, and locations, ensuring they are well-prepared and informed for their upcoming interviews.

  • Frontend: jQuery, HTML5, Bootstrap CSS3,
  • JavaScript : Jquery
  • Backend: .NET Core v6.0 (Razor pages), C#, T-SQL (Microsoft SQL Server), Entity-Framework Core, Services, Sessions, View Components
  • Database: Microsoft SQL Server for efficient data storage and retrieval
  • Interview Schedule View: Centralized dashboard displaying all scheduled interviews with specific details
  • Interview Schedule Create:User can register the interview details.
  • User Authentication: Secure user registration and login functionality to safeguard student data and interview details.

Play Station Website(Front end):

Project Description

The PlayStation Website Replica project is a frontend-focused endeavor aimed at recreating the user interface and design of the official PlayStation website. This project showcases my proficiency in frontend technologies and UI/UX design principles by replicating the look and feel of the PlayStation website's frontend.

  • Frontend: HTML5, CSS3, TypeScript
  • JavaScript : TypeScript
  • Frontend Frameworks/Libraries: Angular 15, Bootstrap
  • Homepage Replica: Recreated the layout and design of the PlayStation website's homepage, including banners, promotional sections, and navigational elements.
  • Product Pages: Emulated product pages for various consoles, games, accessories, and services, maintaining the visual consistency and layout of the original website.
  • Aesthetic Consistency: Matched color schemes, typography, and visual elements to replicate the overall aesthetics of the PlayStation website.
